Data Synchronization Instructions
The Data sync configurations below can be scheduled as tasks.
When run they will trigger the program rsync to syncronize the source directory to the destinations directory.
Excludes are file name patterns which will be excluded from the sync. ie( .git, .*)
Options are to be used by advanced Rsync users to add specific options to the rsync command run.
